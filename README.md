# CREWIT

| "크루(CREW)"와 "하자(DO IT)"을 합쳐, 함께 운동할 사람을 모집하여 활동할 수 있는 커뮤니티

![logo-removebg-preview](/uploads/7f2d882c4354f60b1e4df52634d42073/logo-removebg-preview.png)

## 프로젝트 개요

- 운동 크루 커뮤니티 형성 및 공공 체육 시설 정보 제공 서비스
- 프로젝트 제목: CREWIT
- 프로젝트 기간: 2024.11.11 ~ 2024.11.26
- 참여자: 우연주, 백승민

## 사용 기술 스택

- Spring Boot
- Spring
- Java
- HTML5
- CSS3
- JavaScript
- Vue.js
- MySQL

## 개발 도구

- Spring Tool Suite 3
- VSCode
